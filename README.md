# 💸 Sofiia Finance Bot

Мій перший серйозний проєкт на Python — розумний Телеграм-бот для контролю особистих витрат. Бот допомагає записувати витрати за категоріями та зберігає їх у локальну базу даних.

## 🚀 Основні можливості
* **Запис витрат**: Бот запитує суму та категорію, автоматично додаючи поточну дату.
* **База даних**: Використовується SQLite для надійного зберігання всіх транзакцій.
* **Управління**: Можливість видалити останній помилковий запис прямо з інтерфейсу.
* **Звіти**: Перегляд усіх накопичених витрат у зручному форматі.

## 🛠️ Стек технологій
* **Мова**: Python 3.10+
* **Бібліотеки**: `pyTelegramBotAPI`, `python-dotenv`
* **База даних**: SQLite3
